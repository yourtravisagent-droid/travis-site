Travis Travel Toolkit

Award travel without the spreadsheet. One question — "business class, these dates, this many people" — and Travis returns the options, the math, the surcharges, the alternates. Built on live seats data, transfer-partner graphs, and sweet-spot databases.

Not the gross-fare illusion. Real math: fare minus taxes, minus surcharges, over miles used. Flags BA/LH/OS surcharge traps; highlights clean programs (UA metal, ANA, SQ metal, Aeroplan).
